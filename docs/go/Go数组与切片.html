# 一、 Go 语言数组

数组是一个由固定长度的特定类型元素组成的序列，一个数组可以由零个或多个元素组成。

# 1. 创建数组

数组的长度必须是常量表达式，因为数组的长度需要在编译阶段确定。

var a [3]int // [0, 0, 0]

var a = [3]int{1,2,3} // [1, 2, 3]
var a = [3]int{1,2} // [1, 2, 0]

// 长度位置若是“...”省略号，则数组长度=初始化值个数
q := [...]int{1, 2, 3} // q 长度为3

// 初始化某几个索引位置的元素
r := [...]int{4: -1} // [0, 0, 0, 0, -1]
r := [...]int{2: 2, 4: -1} // [0, 0, 2, 0, -1]

# 2. 读取数组

根据索引读取数组

r := [...]int{2: 2, 4: -1}
a := r[2] // 2
b := r[3] // 0

// i 为索引，v 为索引对应的值
for i, v := range r {
	fmt.Printf("%d %d \n",i ,v)
}
// 不需要索引
for _, v := range r {
	fmt.Printf("%d %d \n",i ,v)
}

# 3. 数组比较

如果一个数组的 元素类型 是可以相互比较的，那么 数组类型 也是可以相互比较的。

直接通过 == 比较运算符来比较两个数组。

只有当两个数组的 所有元素都是相等 的时候，数组才是相等的。

a := [2]int{1, 2}
b := [...]int{1, 2}
c := [2]int{1, 3}
fmt.Println(a == b, a == c, b == c) // "true false false"

# 4. 数组的缺陷

数组的长度是固定的。

然而实际应用中，常常需要增长活收缩序列。

所以更常用的数据类型是 切片。

# 二、切片

Slice（切片）代表变长的序列，序列中每个元素都有相同的类型。

一个 slice 由三个部分构成：指针、长度和容量

# 1. 创建切片

1）由数组切片生成

months := [...]string{1: "January", /* ... */, 12: "December"}

Q2 := months[4:7]
summer := months[6:9]

fmt.Println(Q2) // ["April" "May" "June"]
fmt.Println(summer) // ["June" "July" "August"]
