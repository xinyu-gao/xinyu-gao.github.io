Java 反射

橙子 2020-03-05 Java

# 反射简介

# 1. 定义

反射就是 Reflection，反射是 Java 语言的一种特性，它可以让 Java 程序在运行过程中检查、操作程序的内部属性。例如，它可以让一个 Java 类获取并显示其所有的成员变量名称。

# 2. 目的

官方文档对反射目的的描述：

  • 让开发人员可以通过外部类的全路径名创建对象，并使用这些类，实现一些扩展的功能。
  • 让开发人员可以枚举出类的全部成员，包括构造函数、属性、方法。以帮助开发者写出正确的代码。
  • 测试时可以利用 反射 API 访问类的私有成员，以保证测试代码覆盖率。

可以在不修改代码的情况下，利用外部文件进行对象和方法的更改，有效降低代码耦合性。

# 3. 应用

反射似乎破坏了 封装性，甚至让私有变量都可以让外部访问，失去了一定的安全性。

使用反射，首先代码非常繁琐，其次，它更多地是给工具或者底层框架来使用，目的是在不知道目标实例任何信息的情况下，获取特定字段的值。

Java Beans 就是一个使用反射机制的例子。

# 反射 API

Class 类：表示正在运行的 Java 应用程序中的类的 实例。

Field 类：提供有关类的 属性信息，以及对它的动态访问权限。它是一个封装反射类的属性的类。

Method 类：提供关于类的 方法 的信息，包括抽象方法。它是用来封装反射类方法的一个类。

Constructor 类：提供有关类的 构造方法 的信息，以及对它的动态访问权限。它是一个封装反射类的构造方法的类。

Object 类：Object 是所有 Java 类的 父类。所有对象都默认实现了 Object 类的方法。

# 获取 Class 类

# 1. 原理

JVM 加载每个 class 的时候创建了对应的 Class 实例，并在该实例中保存了该 class 的所有信息。 我们可以通过 Class 实例来获取对应的 class 的所有信息，即 反射。

# 2. 获取 Class 实例

获取 Class 实例 有 3 种方法：

  1. 需要 导入类包，直接通过 class 的静态变量 class 获取：
Class studentClass = Student.class;
1
  1. 需要 完整类名，通过静态方法 Class.forName()（常用）：
Class studentClass = Class.forName("com.***.Student");
1
  1. 需要 实例对象，通过方法 getClass()：
Student student = new Student();
Class studentClass = student.getClass();
1
2

# 3. JVM 动态加载

在执行程序时，只有在用到 class 的时候才加载到内存中。

例子：优先使用某类 A，当 A 不存在时，使用类 B。

# 获取成员变量

# 1. 获取 Field 实例

Java 提供了 4 种方法：

  1. getDeclaredFields() ：获取所有类（不包括父类）的 public、protect、private 字段。

  2. getDeclaredField(String name) ：根据字段名获取类（不包括父类）的 public、protect、private 字段。

  3. getFields()：仅获取 类及其父类所有的 public 字段。

  4. getField(String name)：根据字段名仅获取 类及其父类的 public 字段。

演示代码：

// People.java
public class People {
    public int age;
    protected String name;
    private int height;
}

// Student.java
public class Student extends People {
    public int weight;
    protected int id;
    private int password;

    public static void main(String[] args) throws NoSuchFieldException {
        Class studentClass = Student.class;

        // 输出： public int Student.weight; protected int Student.id; private int Student.password;
        for (Field i : studentClass.getDeclaredFields()) {
            System.out.print(i + "; ");
        }
        System.out.println();

        // 输出：public int Student.weight;
        System.out.println(studentClass.getDeclaredField("weight"));

        // 输出：public int Student.weight; public int People.age;
        for (Field i : studentClass.getFields()) {
            System.out.print(i + "; ");
        }
    }
}

# 2. 获取 Field 信息

一个 Field 对象包含了一个字段的所有信息：

  • getName()：返回字段名称，例如，"name"。
  • getType()：返回字段类型，也是一个 Class 实例，例如，String.class。
  • getModifiers()：返回字段的修饰符，它是一个 int，不同的 bit 表示不同的含义。

# 3. 访问、设置 Field 值

  • Field.setAccessible(true) ：通过此设置可以访问所有（包括 非 public） 的字段。

  • Field.get(Object)：获取指定实例的指定字段的值，参数为指定的实例。

  • Field.set(Object, Object)：设置字段值，其中第一个 Object 参数是指定的实例，第二个 Object 参数是待修改的值。

# 调用方法

# 1. 获取 Method

Java 提供了 4 种方法：

  1. Method[] getDeclaredMethods() ：获取所有类（不包括父类）的 public、protect、private 方法。

  2. Method getDeclaredField(String name, Class) ：根据 方法名、参数 获取类（不包括父类）的 public、protect、private 方法。

  3. Method[] getMethods()：仅获取 类及其父类所有的 public 方法。

  4. Method getMethod(String name, Class)：根据方法名、参数仅获取 类及其父类的 public 方法。

# 2. 获取 Method 信息

  • getName()：返回方法名称，例如："getScore"。
  • getReturnType()：返回方法返回值类型，也是一个 Class 实例，例如：String.class。
  • getParameterTypes()：返回方法的参数类型，是一个 Class 数组，例如：{String.class, int.class}。
  • getModifiers()：返回方法的修饰符，它是一个 int，不同的 bit 表示不同的含义。

# 3. 调用 Method

普通方法： 使用 Method.invoke() 调用 Method，第 1 个参数是对象实例（即在哪个实例上调用该方法）；后面的可变参数要与方法参数一致，否则将报错。

String str = "Hello world";
// 获取 String substring(int) 方法，参数为 int
Method m = String.class.getMethod("substring", int.class);
// 在 str 对象上调用该方法并获取结果:
String r = (String) m.invoke(s, 6);

非 Public 方法：Method.setAccessible(true)允许其调用，再使用 Method.invoke()。

多态：如果存在重写，则调用重写的方法。

# 调用构造方法

# 1. 方法 1（public、无参数）

使用 Class.newInstance()，但只能调用该类的 public 、且 无参数 构造方法。

Person p = Person.class.newInstance();
1

# 2. 方法 2（任意）

为了调用任意的构造方法，可以通过 Class 实例 获取 Constructor 的方法：

  • getConstructor(Class...)：获取某个 public 的 Constructor。
  • getDeclaredConstructor(Class...)：获取某个 Constructor。
  • getConstructors()：获取所有 public 的 Constructor。
  • getDeclaredConstructors()：获取所有 Constructor。

# 继承关系

# 1. 获取父类类型

getSuperclass()：获取当前类的父类的类型。除 Object 外，其他任何非 interface 的 Class 都必定存在一个父类类型。

Class n = String.class.getSuperclass();
1

# 2. 获取 interface

getInterfaces()：获取当前类（不包括父类）的所有实现的接口：

Class[] n = String.class.getInterfaces();
1

# 3. 判断子类

isAssignableFrom()：方法可以判断一个类是否为其子类。

Object.class.isAssignableFrom(Integer.class); // true
