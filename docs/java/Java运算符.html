橙子 2020-03-16 Java

# 基础运算符

算术运算符： + 、-、 * 、/ 、% 表示加、减、 乘、除、求余运算。

关于除法：

  • 两个操作数都是整数时，表整除。否则为浮点除法。
  • 整数除以 0 产生异常
  • 浮点数除以 0 得到无穷大或者 NaN。
System.out.println(10/3); // 3
System.out.println(10.0/3); // 3.3333333333333335
System.out.println(10/0); // java.lang.ArithmeticException: / by zero
System.out.println(10.0/0); // Infinity

关于浮点数运算：

  • 在默认情况下， 虚拟机允许对中间计算结果采用扩展的精度。(不会溢出)
  • 但是， 对于使用 strictfp 关键字标记的方法必须使用严格的浮点计算来生成可再生的结果。（可能溢出）

**赋值二元运算符：**运算符放在 = 号左边，如 += 、 %=

x += 2;
// 等价于
x = x + 2;
1
2
3

**自增自减运算符：**后缀形式 x++、x--；前缀形式 ++x 、--x。

单独使用无区别，但在表达式中， ++x 与 x++ 有区别：

  • 前缀形式 y = ++x：先自增后赋值。
  • 后缀形式 y = ++x：先赋值后自增。

# 数学函数与常量

平方根： y = Math.sqrt(x);（ x 的平方根）。

幂运算： y = Math.pow(x, n); （x 的 n 次幂，三个数都为 double 类型）。

取模运算：

  • % 为取余，取余运算在计算商值向0方向舍弃小数位.
  • floorMod() 为取模，取模运算在计算商值向负无穷方向舍弃小数位（根据欧几里德规则：余数总大于 0）。
System.out.println(-5 % 3);  // -2
System.out.println(floorMod(-5, 3)); // 1
1
2

注意：对于负除数，floorMod() 会得到负数结果，不过实际中很少出现.

三角函数： Math.sin()、cos、tan、atan、atan2 等。

指数函数及其反函数：Math.exp e 的 n次方、Math.log 自然对数、Math.log10 以 10 为底的对数。

常量：Math.PI（Π） 、Math.E （e）。

# 强制类型转化

数值之间不合法转化，需要进行强制转化，可能丢失一些信息。

例如，double 转 int：

double score = 94.67;
int myScore = (int) score; // 94
1
2

这种方法将 94.67 进行了截断。

要想四舍五入，使用方法：(int) Math.round(x);。

注意：如果试图将一个数值从一种类型强制转换为另一种类型， 而又超出了目标类型的 表示范围，结果就会截断成一个完全不同的值。

赋值二元运算符，也可以发生强制转化，是合法的。

x += 3.5;
1

# 其他运算符

boolean 运算符： < 小于、 > 大于、<= 小于等于、 >= 大于等于、== 等于、!= 不等于。

逻辑运算符：&& 逻辑“ 与” 、|| 逻辑“ 或” 、! 逻辑“ 非” 。

三元运算符：condition ? expression1 : expression2。条件为 true 则为第一个表达式的值；否则为第二个表达式的值。

位运算符： & (按位与)、 | (按位或) 、^ (按位异或)、 ~ (按位取反)

<<、>> 将位模式左移或右移（右移时用符号位填充高位）。>>> 用 0 填充高位，没有 <<<。
