# 一、Eureka 注册中心

注意：本文以下每个项目都是以模块的方式创建的，而不是以 new project 方式创建的。（即先创建一个工程目录，在该目录下 new module 的方式创建。）

# 1. 初始化工程

在初始化工程的时候，选择 Spring Cloud Discovery 中的 Eureka Server 依赖：

    <artifactId>spring-cloud-starter-netflix-eureka-server</artifactId>

# 2. 修改启动类

启动类上加注解 @EnableEurekaServer 来启动 Eureka 服务器：

@EnableEurekaServer
@SpringBootApplication
public class EurekaServerApplication {

    public static void main(String[] args) {
        SpringApplication.run(EurekaServerApplication.class, args);
    }

}

# 3. 配置 Eureka

server:
  port: 8201 # 服务端口号

spring:
  application:
    name: eureka-server # 指定服务名称

eureka:
  instance:
    hostname: localhost # 指定主机地址，默认去环境变量中寻找
  client:
    fetch-registry: false # 指定是否要从注册中心获取服务（注册中心不需要开启）
    register-with-eureka: false # 指定是否要注册到注册中心（注册中心不需要开启）
  server:
    enable-self-preservation: false # 禁用自我保护功能

# 2. 修改启动类

在启动类上添加 @EnableDiscoveryClient 注解。

注：有的教程可能会标注 @EnableEurekaClient 注解。该注解只能用在 Eureka 架构上，其他的架构不可以用（例如 Alibaba），但是 @EnableDiscoveryClient 都可以用，应用场景更广泛。

# 3. 修改 yml 配置文件

server:
  port: 8301 # 运行端口号
  
spring:
  application:
    name: eureka-client # 服务名称
    
eureka:
  client:
    register-with-eureka: true # 注册到注册中心
    fetch-registry: true # 获取注册实例列表
    service-url:
      defaultZone: http://localhost:8201/eureka/ # 注册中心地址
