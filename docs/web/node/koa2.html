# 一、Node.js 与 Koa2

# 1. 什么是 Node.js ？

JavaScript 最早运行在浏览器中，但是浏览器只提供了一个上下文，它定义了使用 JavaScript 可以做什么。Node.js 是另一种上下文，它允许脱离浏览器运行 JavaScript 代码。

运行在浏览器中的 JavaScript 的用途是操作DOM，浏览器提供了 document 之类的内置对象。而运行在 Node.js 中的 JavaScript 的用途是操作磁盘文件或搭建HTTP服务器，Node.js 就相应提供了 fs、http 等内置对象。

Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。Node.js 的包管理器 npm，是全球最大的开源库生态系统。

# 2. 什么是 Koa ？

Koa 是一个新的 web 框架， 致力于成为 web 应用和 API 开发领域中的一个更小、更富有表现力、更健壮的基石。

通过利用 async 函数，Koa 会丢弃回调函数，并有力地增强错误处理。

# 3. 安装过程

以 Windows 10 为例：

安装 Node.js：在官网 http://nodejs.cn/download/ 下载 msi 安装包，选择安装路径进行安装。注意：node.js 环境版本保证在 v7.6 以上

检测是否成功：

在控制输入node -v、npm -v，若显示版本说明则成功。

安装 nvm：

https://github.com/coreybutler/nvm-windows/releases 下载安装包。

安装 Koa2：

npm install koa

# 二、有道翻译 API

# 1. 有道 API 介绍

文本翻译 API 接口提供有道的翻译服务，包含了中英翻译和小语种翻译功 能。您只需要通过调用文本翻译 API，传入待翻译的内容，并指定要翻译的源 语言（支持源语言语种自动检测）和目标语言种类，就可以得到相应的翻译结 果。

文本翻译 API HTTPS 地址：https://openapi.youdao.com/api 。

请求的传输方式为 HTTPS、请求方式为 GET/POST、字符编码为 UTF-8、 请求格式为表单、相应格式为 JSON。

# 2. 获取 ID 和密钥

登录有道智云平台后，点击“应用管理” -> “我的应用” -> “创建应 用”，根据提示信息，完成应用创建。在创建应用时请选择 “API”， 应用创建成功后获取应用 ID（appKey）和应用密钥等信息。

点击“应用管理” -> “我的应用”，在应用列表找到需要绑定服务的应 用，点击“绑定服务。

获取应用 ID 和应用密钥。

/**
 * 通过一个人的 id 找到和他身材相似的人的 id
 * @param {*} db  数据库实例
 * @param {*} id  
 */
function findSimilarById(db, id) {
    // get 为查询一条数据
    db.get("select * from student where ID = ?", id, function (err, row) { 
        if (err) {
            global.result = err; // 返回异常
        } else {
            let person1 = row; //返回该人的信息
            // 查询其他人的信息，all查询多条信息
            db.all("select * from student where ID != ?", id, function (err, row) { 
                if (err) {
                    global.result = err; // 返回异常
                } else {
                    otherPersons= row; //返回id
                    let data_min  = 100000;
                    for (i in otherPersons){  // 遍历所有其他人的信息，对比找出差距最小的
                        data2 =  Math.sqrt((person1.JK- otherPersons[i].JK)**2+ // 计算体型差
                        (person1.XW- otherPersons[i].XW)**2+
                        (person1.YW- otherPersons[i].YW)**2+
                        (person1.XC- otherPersons[i].XC)**2+
                        (person1.XM- otherPersons[i].XM)**2+
                        (person1.SG- otherPersons[i].SG)**2+
                        (person1.TZ- otherPersons[i].TZ)**2)
                        if(data_min < data2 ){
                            data_min = data_min;
                        }else{  // 若比之前的小，则替换最小体型差值，和id名
                            data_min = data2;
                            global.result = otherPersons[i].ID // 替换结果id
                        }
                    }
                }
            });
        }
    });
    return;
}
/**
 * 查找 student 表中所有成员的数据，返回 JSON
 * @param {*} db 数据库实例
 */
function queryAll(db) {
    //select 语句
    db.all("select * from student ", function (err, row) {
        if (err) {
            global.result = err; // 返回异常
        } else {
            global.result = JSON.stringify(row); // 转化 json
        }
    });
    return;
}
/**
 * 插入某一个人的数据到 student 表
 * @param {*} db 数据库实例
 * @param {*} postData 数据集
 */
function insertStudent(db, postData) {
    db.run(`insert into student values (?,?,?,?,?,?,?,?,?,?,?)`, // 11 个数据
        [
            postData.ID, postData.NAME, postData.GENDER, postData.AGE,
            postData.JK, postData.XW, postData.YW, postData.XC,
            postData.XM, postData.SG, postData.TZ
        ],
        function (err) {
            if (err) {
                global.result = err; // 返回异常
            } else {
                global.result = "Insert Data Success!"; // 返回插入成功的信息
            }
        });
    return;
}

处理请求

app.use(bodyParser()) // 使用ctx.body解析中间件
app.use(async (ctx) => {

    if (ctx.url.indexOf("/query/similar?id=") == 0) { // 处理查找相似身材的人的id请求，get
        let id = ctx.request.query.id; // 获取 id
        findSimilarById(db, id); // 调用函数
        ctx.body = global.result; // 返回结果
    }
    else if (ctx.url.indexOf("/queryAll") == 0) { // 处理查找所有成员信息的请求，get
        queryAll(db); // 调用函数
        ctx.body = global.result; // 返回结果
    }
    else if (ctx.url.indexOf("/insert/student") == 0) { // 处理插入数据请求，post
        let postData = ctx.request.body; // 获取 post 的 body 中的信息
        insertStudent(db, postData); // 调用函数
        ctx.body = global.result; // 返回结果
    } else {
        ctx.body = 404; // 其他请求返回 404
    }
});
