Vue 使用 axios

橙子 2020-12-01 Vue

# axios介绍

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。

# axios安装

推荐使用 cnpm 代理： cnpm install axios

# axios使用

# 1. 引用 axios：

在 main.js 中写入

import axios from 'axios';
Vue.prototype.$axios = axios //全局注册，使用方法为:this.$axios

2

# 2. 发送请求

在代码中使用 axios 发送请求：

this.$axios({
      method: "post",
      url: "apiURL", // 接口地址
      data: {
        keyword: "1"   // 传接口参数
      }
    })
      .then(res => {
        console.log(res, "success");   // 成功的返回      
      })
      .catch(error => console.log(error, "error")); // 失败的返回

# 跨域处理

跨域一般使用 proxyTable 或者 Nginx 来解决，但本质都是使用代理。前者一般在调试开发时使用， 后者在生产环境中使用（在此仅描述前者的使用方法）。

# 1. 修改配置文件

在根目录下创建 vue.config.js 文件（如果没有的话），写入：

module.exports = {
    devServer: {
      proxy: {
        '/api': {
          target: 'http://localhost:8081', // 要请求的地址
          ws: true,
          changeOrigin: true,// 是否开启跨域
          pathRewrite: {
            '^/api': ''
          }
        }
      }
    }
  };

# 封装示例

为了避免每次请求时都去判断各种各样的网络情况，我们应对 axios 进行封装。下面介绍简单的封装示例。

# 1. 请求和响应拦截

在 src 目录下新建 axisoRequest.js 文件，这样就不用全局引入 axios 了，避免 main.js 过于臃肿，也可以使项目可以按需引入：

import axios from 'axios'
import { Message } from 'element-ui'
import { getToken } from './auth'
const creatRequest = axios.create({
    // 请求地址
    baseURL: process.env.VUE_APP_BASE_API,
    // 请求超时时间
    timeout: 7000
})

creatRequest.interceptors.request.use(config => {
    // console.log('------request------' + JSON.stringify(config))
    // 设置请求头 ，可选
    config.headers = {
        // 请求格式为 json
        'Content-Type': 'application/json',
        // 设置 Token
        'Authorization': getToken()
    }
    // 将数据转化为 json 格式
    config.data = JSON.stringify(config.data)
    return Promise.resolve(config)
}, error => {
    Message.error({ message: '请求超时!' })
    return Promise.reject(error)
})

creatRequest.interceptors.response.use(data => {
    // console.log('------response------', getToken(), data)
    // 将 data 值取出，默认值： {}
    data = data && data.data || {}
    // 返回 200 代码，并且 data 字段存在，正常返回 data，否则抛出异常
    if (data && data.code === 200 && data.data) {
        return Promise.resolve(data.data)
    } else {
        return Promise.reject(data.message)
    }
}, err => {
    return Promise.reject(err)
})

export default creatRequest

    return creatRequest({
        url: '/login',
        method: 'post',
        data: data
    })
